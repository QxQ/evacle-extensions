about

this repository contains the source code for all of the extensions evacle has made. you can download any of the extensions here.

extensions inside this repository

pinner - adds a pin tab button so you don't have to right click to pin or unpin.

close pinned tab - puts a close tab button in the address bar when you are currently viewing a pinned tab.

live on - keeps chrome from closing when the last tab is closed by creating a pinned tab when there is one tab left.

undo close - allows you to undo closing a tab.

unload - unload a tab. great for slow computers.

quick start - start chrome with all extenstions disabled, click to enabled some. great for slow computers.

connection notifier - tells you if you don't have internet connection.

reload once connected - asks you if you wan't to reload all of your pages once chrome finds internet

window manager - lets you manage your windows in the click of a button. comparable to firefox's panorama
